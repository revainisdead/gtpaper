
### Installation
Refer to `nix_commands.txt`

### Use a virtual environment
`sudo apt-get install python3-venv`
`mkdir gtpaper-venv`
`python3 -m venv gtpaper-venv`
`cd gtpaper-venv`
`source bin/activate`
`sudo pip3 install requirements.txt`

### Run
##### [Backend] Django
`./start_api.sh`
##### [Frontend] React
`cd client`
`npm start`
